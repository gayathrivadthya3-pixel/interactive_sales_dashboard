# Interactive Sales Dashboard

## 📌 Project Overview

The **Interactive Sales Dashboard** is a data visualization project developed using **Python**, **Seaborn**, **Matplotlib**, and **Plotly**.
The dashboard provides insights into sales performance, product analysis, customer trends, and regional distribution through professional statistical and interactive visualizations.

This project demonstrates:

* Data preprocessing
* Statistical analysis
* Dashboard design
* Interactive visualizations
* Clean coding principles

---

# 🚀 Features

## ✅ Statistical Visualizations

* Box Plots
* Violin Plots
* Histograms
* Correlation Heatmaps

## ✅ Interactive Dashboard

* Interactive Sales Trends
* Product Distribution Pie Charts
* Regional Performance Analysis
* Hover Effects and Dynamic Charts
